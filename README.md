# MyAigoprov - Διαχείριση Κτηνοτροφικής Μονάδας  

**Έκδοη 1.0**

Το **MyAigoprov** είναι μια εφαρμογή Android που απευθύνεται σε κτηνοτρόφους, επιτρέποντάς τους να διαχειρίζονται εύκολα τα ζώα της μονάδας τους. Οι χρήστες μπορούν να προσθέσουν, να επεξεργαστούν και να διαγράψουν ζώα, αποθηκεύοντας βασικές πληροφορίες όπως αριθμός σήμανσης, γένος και έτος γέννησης. Επίσης, περιλαμβάνει λειτουργία καταγραφής της παραγωγής γάλακτος.  

## 🔹 Τεχνολογίες  
- **Γλώσσα Προγραμματισμού**: Java  
- **Περιβάλλον Ανάπτυξης**: Android Studio  
- **UI**: XML Layouts  
- **Βάση Δεδομένων**: SQLite  

## 🎯 Βασικές Λειτουργίες  
- **Καταγραφή Ζώων** με μοναδικό αριθμό σήμανσης  
- **Διαχείριση δεδομένων** (προσθήκη, επεξεργασία, διαγραφή)  
- **Διαχωρισμός Ζώων** σε αίγες και πρόβατα  
- **Καταγραφή Παραγωγής Γάλακτος** ανά ημέρα  
- **Offline αποθήκευση δεδομένων** με SQLite
