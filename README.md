# MyAigoprov - Διαχείριση Κτηνοτροφικής Μονάδας  

Το **MyAigoprov** είναι μια εφαρμογή Android σχεδιασμένη για κτηνοτρόφους, επιτρέποντάς τους να διαχειρίζονται εύκολα τα ζώα της μονάδας τους. Υποστηρίζει την προσθήκη, επεξεργασία και διαγραφή ζώων, με καταγραφή βασικών πληροφοριών όπως αριθμός σήμανσης, γένος και έτος γέννησης.  

## 🔹 Τεχνολογίες  
- **Γλώσσα Προγραμματισμού**: Kotlin  
- **Περιβάλλον Ανάπτυξης**: Android Studio  
- **UI Framework**: Jetpack Compose  
- **Βάση Δεδομένων**: SQLite  

## 🎯 Βασικές Λειτουργίες  
- **Καταγραφή Ζώων** με μοναδικό αριθμό σήμανσης  
- **Διαχείριση δεδομένων** (προσθήκη, ενημέρωση, διαγραφή)  
- **Διαχωρισμός Ζώων** σε αίγες και πρόβατα  
- **Offline αποθήκευση δεδομένων** με SQLite
